# 📌 Project Overview

This project focuses on analyzing e-commerce order and product data to uncover sales trends, customer behavior, delivery performance, and product demand patterns.

The analysis was performed using Google Sheets and Looker Studio to transform raw transactional data into interactive dashboards and actionable business insights.

---

# 🛠 Tech Stack

### Google Sheets

* Data Cleaning
* Pivot Tables
* Pivot Charts
* Data Validation
* Conditional Formatting
* Lookup Functions
* SUMIFS & COUNTIFS
* Dashboard Preparation

### Looker Studio

* Interactive Dashboards
* KPI Cards
* Filters
* Product Analysis
* Regional Sales Analysis
* Order Status Tracking
* Business Reporting
