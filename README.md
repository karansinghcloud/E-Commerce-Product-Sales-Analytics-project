# 📌 Project Overview

This project focuses on analyzing e-commerce order and product data to uncover sales trends, customer behavior, delivery performance, and product profitability.

The analysis was performed using Advanced Excel and Tableau to transform raw transactional data into interactive dashboards and business insights.

---

# 🎯 Business Problem

E-commerce businesses generate large volumes of operational data every day. Without proper analysis, it becomes difficult to identify:

- Best-selling products
- High return products
- Regional sales trends
- Delivery performance
- Customer demand patterns
- Order cancellation reasons
- Revenue opportunities

This project helps management make data-driven decisions.
